## PAIMAI UI 组件库

### 本地预览

1.首先执行，安装依赖命令
```
npm install 
```
2.安装完成后执行

 ```
 npm run storybook
 ```

3.本地可以看预览后，按照已有组建封装规则，封装组建，并且创建.stories.tsx

### 组件使用步骤
```
jnpm install @paimai/components-ts
```

### 使用

```
import { Button } from '@paimai/components-ts'
```
